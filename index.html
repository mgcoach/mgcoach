Envie o plano para o paciente via WhatsApp em formato HTML interativo, ou exporte como PDF/JSON.
📲 RECOMENDADO: Salvar para WhatsApp (HTML)
O arquivo HTML preserva todas as funcionalidades — substituições clicáveis, ciclagem de carbos (3 dias), divisão por refeição, e funciona perfeitamente no celular do paciente via WhatsApp.

Como usar:
1. Clique em "📲 Salvar para WhatsApp (HTML)"
2. O arquivo .html é baixado
3. Anexe no WhatsApp do paciente (📎 → Documento)
4. Paciente toca no arquivo → abre no celular
5. Pode tocar nas substituições e ver alternativas!
